States API Index Page

REST API available at /states
The following routes are supported:

  • /states [GET] (Provides access to all state data)
  • /states/:state [GET] (Provides access to a specific state)
  • /states/:state/funfact [GET, POST, PATCH, DELETE] (Provides access to a specific state's fun facts)
  • /states/:state/:property [GET] (Provides access to a specific state's specific properties)
